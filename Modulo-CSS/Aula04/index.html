Box Model: O modelo de caixa é um conceito fundamental em CSS que descreve como os elementos HTML são representados visualmente na página. Ele é composto por quatro partes principais: conteúdo, preenchimento (padding), borda (border) e margem (margin). Cada uma dessas partes tem suas próprias propriedades e pode ser personalizada para controlar o layout e a aparência dos elementos.
