Cores e Fontes

Existem 4 maneiras de usar cores em CSS:

  • Nome da cor (red, blue, green)
  • Valor hexadecimal:

    #FF0000

    #0000FF

    #008000

  • Valor RGB:

    rgb(255, 0, 0)

    rgb(0, 0, 255)

    rgb(0, 128, 0)

  • Valor RGBA:

    rgba(255, 0, 0, 0.5)

    rgba(0, 0, 255, 0.5)

    rgba(0, 128, 0, 0.5)
